# Nodetchat

Projet de base pour démarrer un chat avec Express et Node.js

## Installation

Clonez le repo :

`git clone https://github.com/jmpp/nodechat.git`

Installez les dépendances :

`npm install`

## Démarrage

Lancez un serveur :

`npm run dev`

... puis naviguez sur `http://localhost:3000/`
